# Format Templates

Standard structures for common document types.

## Paper

```
1. Introduction
   - Hook / motivation
   - Gap in knowledge
   - This paper's contribution
   - Roadmap

2. Background / Related Work
   - Prior approaches
   - What's missing

3. Methods / Approach
   - Description of method
   - Justification of choices

4. Results
   - Findings organized logically
   - Figures/tables as needed

5. Discussion
   - Interpretation
   - Limitations
   - Future work

6. Conclusion
   - Summary of contribution
   - Broader implications
```

## Grant Proposal

```
1. Specific Aims (1 page)
   - Opening paragraph (significance)
   - Aim 1: <title>
   - Aim 2: <title>
   - Aim 3: <title>
   - Impact statement

2. Significance
   - Importance of problem
   - How this advances the field

3. Innovation
   - What's new about this approach
   - Why it will succeed

4. Approach
   - Preliminary data
   - Research plan by aim
   - Timeline
   - Potential pitfalls and alternatives
```

## Slides / Presentation

```
1. Title slide
   - Title, author, affiliation, date

2. Motivation
   - Why should audience care?
   - 1-2 slides max

3. Problem statement
   - Clear, specific problem

4. Key insight / contribution
   - The "aha" moment

5. Evidence
   - 2-4 slides supporting the claim
   - One idea per slide

6. Implications
   - What does this mean?

7. Conclusion / takeaway
   - 1 slide, memorable summary

8. (Optional) Questions / discussion
```

## Lecture Notes

```
1. Learning objectives
   - What students will be able to do

2. Motivation / context
   - Why this matters
   - How it connects to prior material

3. Core concepts
   - Chunked into digestible pieces
   - Each concept gets:
     - Definition
     - Intuition
     - Example

4. Worked examples
   - Step by step
   - Common variations

5. Common misconceptions
   - What students get wrong
   - Why

6. Summary
   - Key takeaways
   - Connections to next topic

7. Exercises / problems
   - Practice opportunities
   - Range of difficulty
```
